# Proyecto Cohabit

## Propósito

El propósito del proyecto es crear una aplicación web que facilite la gestión de recursos compartidos en entornos compartidos, permitiendo a los compañeros de un espacio común reservar y organizar el uso de recursos como cocina, baño, salón o lavadora, evitando conflictos y esperas, y promoviendo una convivencia más ordenada y transparente.
